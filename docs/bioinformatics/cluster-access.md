# Cluster access

This page explains how to get an account on the **Farm** (CAES) cluster and set up SSH access.

---

## 1. Request an account

1. Go to the UC Davis HPC account request forms:  
   <https://hpc.ucdavis.edu/account-request-forms>

2. On that page, look for the text:  
   > “Please create a request on HiPPO to request an account on Farm, Franklin, and Hive.”

3. Click the **HiPPO** link and submit an account request:
   - Select the **Farm / CAES cluster**.
   - Choose **“Monroe, Grey”** as your **sponsor**.

---

## 2. Set up SSH access

While submitting your request (and/or in the follow-up instructions from HPC):

1. Follow the **SSH key setup** instructions provided on the HPC site.
2. Generate an SSH key pair on your local machine (if you don’t already have one).
3. Upload / register your **public key** with the HPC system as described.
4. Once your account is approved, you can log in to Farm using that SSH key.

> If you get stuck at any step (HiPPO, sponsor selection, or SSH keys), ask in the lab Slack or contact Grey.
