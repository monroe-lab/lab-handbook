## Reagents needed for Module 1 (planting + crossing)

Check inventory for:
- [ ] MS basal salt mixture
- [ ] MES buffer
- [ ] TC Agar (A175)
- [ ] Sucrose (tissue culture grade or molecular biology grade)
- [ ] KOH (5N and 0.5N stocks)
- [ ] 70% ethanol (for seed sterilization)
- [ ] Sodium hypochlorite / bleach
- [ ] Triton X-100 (20% stock)
- [ ] Petri dishes (100 mm, sterile)
- [ ] Surgical tape (Micropore)
- [ ] Potting soil (Arabidopsis-appropriate)
- [ ] Small pots / flats
