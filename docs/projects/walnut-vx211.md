# Walnut VX211

Short-read WGS of clonal VX211 walnut rootstock trees, including off-type / variant phenotypes. VX211 is one of the dominant clonal "Paradox" walnut rootstocks (alongside Vlach), increasingly important to California walnut. The accessions here are an 8-sample N/W (north / west position) panel collected by [[Matt Davis]] on 2026-03-28 and queued for sequencing once the [[kerman-somatic|Kerman Somatic]] pipeline finishes.

## Summary

VX211 is a clonal Paradox-type walnut rootstock. As with pistachio's UCB-1, every VX211 tree in production traces back to a single original individual through grafting, which makes any phenotypic off-types in commercial plantings interesting from a somatic-mutation perspective: are they real heritable variants, layer-specific somatic mutations, or sampling artifacts?

This project is the first probe at that question. [[Matt Davis]]'s 2026-03-28 collection trip yielded 8 leaf samples spanning rows 15-16 (`R15`-`R16`), blocks 1-3 (`B1`-`B3`), positions N/W (north/west). All 8 are filed as Illumina WGS, `Tissue collected — to be completed after the Kerman Somatic project is complete`.

This is the highest-priority of Matt's three new walnut sample sets ([[walnut-vx211|VX211]] → [[walnut-cut-leaf|Cut leaf]] → [[walnut-weeping-phenotype|Weeping]] is the priority order Matt set with the lab pre-Europe).
