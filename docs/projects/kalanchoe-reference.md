# Kalanchoe Reference

PacBio HiFi de novo reference assembly of *Kalanchoe* (likely *K. tubiflora*) — supporting Matt Davis's undergraduate-led project to establish *K. tubiflora* as a model organism for somatic mutation in clonally propagated plants.

## Summary

*Kalanchoe tubiflora* is small, fast-growing, and exclusively asexual, which makes it an attractive model for studying somatic mutation accumulation in clonal lineages. The broader Kalanchoe Model System project (led by [[Matt Davis]] and undergraduate researchers under a Jastro-Shields 2024 award) plans HMW DNA extraction, PacBio HiFi sequencing, and hifiasm haplotype-phased assembly with manual curation. This "Kalanchoe Reference" project is the sequencing/assembly arm of that effort: produce the reference genome that downstream somatic-mutation work will be mapped to.

Status as of 2026-04-29: tissue collection still pending. One placeholder concept-level accession card is filed; per-tree / per-line cards should be created when material arrives in the lab.
