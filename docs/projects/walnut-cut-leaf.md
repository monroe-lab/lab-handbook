# Walnut Cut Leaf

Short-read WGS of a walnut cut-leaf phenotype tree (*Juglans regia* × *J. laciniata*) — one of three walnut phenotype-tracking projects [[Matt Davis]] sampled on 2026-03-28. The cut-leaf phenotype has historical provenance traced back to 1877 (the source of cut-leaf nigra is Harry Lagerstadt's filbert breeding program at Oregon State, per Chuck Leslie's archival records).

## Summary

This is one of a small set of walnut variant-phenotype sequencing projects ([[walnut-cut-leaf|Cut Leaf]], [[walnut-vx211|VX211]], [[walnut-weeping-phenotype|Weeping]]) that share a single field collection trip (Matt Davis, 2026-03-28) and a shared sample-prep timeline (extraction begins after the [[kerman-somatic|Kerman Somatic]] pipeline finishes; Matt's priority order on return from Europe is VX211 → Cut leaf → Weeping). One sample is filed under this project: `R1_T7_RxL`, an Illumina WGS sample of the cut-leaf phenotype tree (`RxL` = *regia* × *laciniata*).

The genomic question is whether the cut-leaf phenotype has a heritable somatic / structural origin — a single sample is a probe, not a population genetics study, but it gives a starting point for design.
