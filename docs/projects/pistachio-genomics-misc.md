# Pistachio Genomics

Catch-all bucket for *Pistacia vera* genomics accessions that don't fit cleanly into one of the named pistachio projects ([[pistachio-pangenome|Pistachio Pangenome]], [[pistachio-wolfskill-collection|Wolfskill]], [[kerman-somatic|Kerman Somatic]], [[pbts|PBTS]], [[pistachio-hybrids|Hybrids]], [[pistachio-short-read-wgs|Short-Read WGS]]). Use this for one-off samples — single-tree experiments, troubleshooting / methods-development extractions, surplus tissue from other workflows.

## Summary

This is a catch-all project. As accessions accumulate here over time, they should be migrated into the appropriate named pistachio project where one exists. Each sample card filed here should make clear in its `status_note` why it lives outside the named projects.

Currently one accession is filed under this project: `PIST-4`, an extracted *Pistacia vera* HiFi sample (assigned to `J. Freckles` for QC / fun-test purposes).
