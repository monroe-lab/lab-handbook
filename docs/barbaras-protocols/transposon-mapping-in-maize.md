# Transposon Mapping in Maize

*Created by greymonroe on 2026-04-12*

Transposable elements, or "jumping genes," were first discovered in maize by Barbara McClintock in the 1940s. This protocol covers the detection of Ac/Ds transposon activity using kernel pigmentation assays.
