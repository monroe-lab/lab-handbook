# Lab Notebooks

Each lab member maintains a notebook in this section. Entries follow the [notebook template](template.md) — one entry per day in the lab.

## How to Use

1. Create a folder with your name (e.g., `notebooks/your-name/`)
2. Copy the [daily entry template](template.md) for each day
3. Name files by date: `2026-04-02.md`
4. Fill in your activities, results, and reflections
5. Commit via the [Edit Wiki](https://monroe-lab.github.io/lab-handbook/editor/) or git
