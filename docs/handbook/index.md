# Lab Handbook — General Documents

This section holds general lab documentation that doesn't fit a more specific category: lab policies, bureaucratic procedures, building access, key procedures, onboarding checklists, and other reference material.

These documents are browsable from the **Documents** tab alongside every other markdown file in the lab handbook.

## Suggested topics

- How to get keys / building access
- Lab onboarding checklist
- Lab policies (lab hours, equipment booking, safety expectations)
- Reimbursement and purchasing procedures
- Travel reimbursement
- Conference attendance policy
- Authorship guidelines
- Data management and backup policy
- Lab meeting expectations

Add new files alongside this one. Anything you drop in `docs/handbook/` will appear in the Documents browser automatically.
