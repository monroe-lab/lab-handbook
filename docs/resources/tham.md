# Tham

> ℹ️ **Chemistry**
> Trade/clinical name for tris(hydroxymethyl)aminomethane — same molecule as [[trizma-base|Trizma base]] and the "Tris" in TAE/TBE/TE — a weak base with pKa ~8.1 at 25°C.

> 💡 **Lab use**
> Workhorse buffer for slightly basic enzymatic work and nucleic-acid storage; the same powder you'd label "Tris base."

> ⚠️ **Safety**
> Low hazard; mild eye and skin irritant. Note that pKa shifts ~0.028 per °C, so titrate at use temperature.
