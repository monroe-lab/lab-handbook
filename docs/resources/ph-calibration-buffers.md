# pH Calibration Buffers

Pre-made buffer solutions of known pH (4.0, 7.0, 10.0) used to calibrate the [[ph-meter]] before each use session. Calibration ensures accurate pH readings.

- **pH 4.0** (red): potassium hydrogen phthalate
- **pH 7.0** (yellow/green): potassium dihydrogen phosphate / disodium hydrogen phosphate
- **pH 10.0** (blue): borate buffer

Store at room temperature. Replace when expired or contaminated (discard if cloudy or if the bottle has been cross-contaminated by dipping a dirty electrode in it).

Used in [[ph-measurement]].
