# Qubit dsDNA HS Assay Kit

High-sensitivity dsDNA quantification kit for the [[qubit-fluorometer]]. Used for input QC in [[ot2-hmw-shearing]] and post-cleanup quant in [[nbd114-multiplexed-flongle-prep]] / [[ot2-automated-nbd114-prep]]. Note the lab Qubit reads ~half of the Genome Center Qubit; cross-check with BR before normalizing for shearing.
