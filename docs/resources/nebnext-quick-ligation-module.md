# NEBNext Quick Ligation Module (E6056S)

Standalone NEB Quick T4 Ligase module (~$200). Provides the Quick T4 ligase used in the sequencing adapter ligation step (step 7) of the NBD114.24 workflow ([[nbd114-multiplexed-flongle-prep]], [[ot2-automated-nbd114-prep]]).
