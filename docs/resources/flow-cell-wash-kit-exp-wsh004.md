# Flow Cell Wash Kit (EXP-WSH004)

Optional flow cell wash kit (~$300, 6 washes). Allows reuse of [[flongle-flow-cells-flo-flg114]] when post-run pore count is still healthy. See [[flongle-sequencing-and-analysis]] § Cleaning up the Flongle. Not strictly required for the [[in-house-hifi-shearing-pipeline]] — single-use is the default — but useful for cost reduction.
