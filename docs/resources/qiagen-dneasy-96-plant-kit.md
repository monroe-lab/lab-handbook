# Qiagen DNeasy 96 Plant Kit

> ℹ️ **Chemistry**
> Silica-column 96-well plant genomic DNA extraction kit. 2 × 96 preps per box per catalog (some list as 4 × 96 = 384).

> 💡 **Lab use**
> Primary 96-well DNA extraction format for [[low-pass-1000-sample-sequencing]]. Grey has the DNeasy Plant **mini** kit (single-tube) in stock; the 96-well format is a different SKU and must be ordered separately.

> ⚠️ **Safety**
> Contains guanidine-HCl and ethanol wash buffers — chaotropic and flammable. Use in fume hood.

Size the order to cover ~2000 extractions with overhead — probably 5-6 boxes depending on which pack size Fisher stocks.

**Procurement:** Fisher punch-out (or VWR punch-out). Vendor: Qiagen. Catalog: 69181.
