# Quant-iT PicoGreen dsDNA Assay Kit

> ℹ️ **Chemistry**
> Ultra-sensitive fluorescent dye for dsDNA quantification; plate-reader format (10 uL samples in 96-well).

> 💡 **Lab use**
> Bulk plate-based dsDNA quantification for 2000-sample scale in [[low-pass-1000-sample-sequencing]]. Qubit HS is lower-throughput but more robust; PicoGreen scales to whole plates.

> ⚠️ **Safety**
> Mild irritant; keep solutions dark to prevent dye photobleaching.

One kit covers thousands of measurements when run in 96-well plates.
