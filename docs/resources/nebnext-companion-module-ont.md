# NEBNext Companion Module for Oxford Nanopore Ligation Sequencing (E7180S)

Bundled NEB enzyme module required by the NBD114.24 ligation workflow. Contains NEBNext Ultra II End Prep and Blunt/TA Ligase modules sized for 24 reactions (~$650).

Required by [[nbd114-multiplexed-flongle-prep]] and [[ot2-automated-nbd114-prep]] because ONT does not ship the end-prep and ligase enzymes in [[sqk-nbd114-24]] itself.

**Verify lab does not already have this before ordering — check freezer inventory.** If the standalone NEB modules are already on hand, skip this bundle and use them instead. Standalone alternatives: [[nebnext-ultra-ii-end-prep]], [[nebnext-quick-ligation-module]], [[neb-blunt-ta-ligase-master-mix]].
