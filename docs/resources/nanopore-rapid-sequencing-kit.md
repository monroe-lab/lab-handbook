# Oxford Nanopore Rapid Sequencing Kit

Kit for preparing nanopore sequencing libraries using a transposase-based "tagmentation" approach. Fragments DNA and attaches sequencing adapters in a single step. Minimal hands-on time (~10-15 min). Preserves long reads from HMW DNA input.

Current version: SQK-RAD114 (check Oxford Nanopore for latest).

**Needs ordering — not yet in the lab.**

Contains: Fragmentation Mix (FRA), Rapid Adapter (RAP), and associated buffers.

Used in [[nanopore-rapid-library-prep]]. Store at -20C.
