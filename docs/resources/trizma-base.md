# Trizma base

> ℹ️ **Chemistry**
> Sigma's brand name for tris(hydroxymethyl)aminomethane, a weak base with pKa ~8.1 at 25°C; the unprotonated "free base" form (vs. Tris-HCl).

> 💡 **Lab use**
> Workhorse buffer for slightly basic enzymatic and nucleic-acid work — TAE, TBE, TE, Laemmli, Western transfer all start here. Adjust pH with HCl at the temperature you'll actually use it.

> ⚠️ **Safety**
> Low hazard; mild eye and skin irritant. Note that pKa shifts ~0.028 per °C, so titrating warm buffer for cold-room use will be off.
