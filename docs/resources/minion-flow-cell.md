# MinION Flow Cell

Nanopore sequencing flow cell for the [[minion-mk1b]]. Contains an array of protein nanopores embedded in an electrical membrane. Current version: R10.4.1.

A new flow cell should have >800 active pores (verified via Platform QC in MinKNOW). Produces 10-50 Gb per run depending on library quality and run duration.

**Needs ordering — not yet in the lab.**

Store at 4C. Use within the expiration date. See [[loading-a-minion-flow-cell]].
