# KAPA Library Quantification Kit (Illumina, qPCR)

> ℹ️ **Chemistry**
> qPCR-based quantification of adapter-ligated library molecules using P5/P7 primers and a standard curve.

> 💡 **Lab use**
> Precise library molarity before pooling for Illumina sequencing in [[low-pass-1000-sample-sequencing]]. More accurate than Qubit for adapter-containing libraries.

> ⚠️ **Safety**
> Low hazard; -20 °C.

500-reaction kit.
