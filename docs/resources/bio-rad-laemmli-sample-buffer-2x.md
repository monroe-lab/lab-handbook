# Laemmli Sample Buffer, 2×

> ℹ️ **Chemistry**
> 2× denaturing/loading sample buffer with bromophenol blue, SDS, and glycerol. Add BME before use.

> 💡 **Lab use**
> Prep Tn5 fractions for SDS-PAGE loading in [[low-pass-1000-sample-sequencing]].

> ⚠️ **Safety**
> Contains SDS and trace BME; low hazard.

30 mL — lasts a long time.
