# Qubit dsDNA BR Assay Kit

Broad Range fluorometric assay kit for quantifying double-stranded DNA. Detection range: 2-1,000 ng/uL in the assay tube. Use for most routine measurements: post-extraction QC, pre-library-prep quantification.

Includes: Qubit dsDNA BR reagent (200x concentrate), Qubit dsDNA BR buffer, Standard 1, Standard 2.

Complementary to the [[qubit-dsdna-hs-assay-kit]] (High Sensitivity, for low-concentration samples).

Used with [[qubit-assay-tubes]] on the [[qubit-fluorometer]]. See [[quantifying-dna-qubit]].

Store at room temperature (reagent) / 4C after opening.
