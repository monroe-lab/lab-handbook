# Welcome! Checklist for new lab members

Slack

- Join slack channel (http://tgac-lab.slack.com/)

Safety:

- Fundamentals of lab safety. [https://safetyservices.ucdavis.edu/training/laboratory-safety](https://safetyservices.ucdavis.edu/training/laboratory-safety)
- Cryogen safety [https://safetyservices.ucdavis.edu/training/cryogen-safety](https://safetyservices.ucdavis.edu/training/cryogen-safety)
- Greenhouse safety. [https://greenhouses.caes.ucdavis.edu/greenhouse-safety](https://greenhouses.caes.ucdavis.edu/greenhouse-safety)
- Safety training sheet signed
- SOPs signed
- Injury and Illness Prevention Plan signed

Server

- Farm Account - If you are going to do high performance computing work, request Form ([https://wiki.cse.ucdavis.edu/cgi-bin/index2.pl](https://wiki.cse.ucdavis.edu/cgi-bin/index2.pl)), follow the instructions for setting up the ssh, select the Farm/CAES cluster and choose "Monroe, Grey" as the sponsor.

Keys - email [pls-keyrequests@ou.ad3.ucdavis.edu](mailto:pls-keyrequests@ou.ad3.ucdavis.edu)

Metal keys:

- Asmundson 11 (Walk-in [[growth-chamber]]), Robbins 262 (Office space), Robbins main (Building), Plant and Environmental Science main (Mailroom code = 10681), Orchard park 307  (Greenhouse space)

Keycards:

- Robbins 164, 166, 168, 170, 172 (lab spaces), Controlled Environments Facility A (growth chambers)

Schedule weekly 1-on-1 meetings

- Send Grey your schedule/weekly availability so we can set up a weekly meeting (and use to pick a lab meeting)
- Create and share a Google Doc where we will keep track of meeting notes (basically just a blank page where we can write out questions, ideas, goals, etc)

Benchling (optional)

- Create Benchling (benchling.com) account - send user name to Grey so you can be added to TGCA projects

Photo and description

- Send a photo for the lab webpage
- Send a short (2-4 sentences) bio about background, interests, etc.
