# Lab-Side Prep — Before Each Cohort

What **the lab** does to make sure students can hit the ground running. Owned by Kehan with Grey input.

Run this ~6-8 weeks before the next cohort's Day 1.

## ~8 weeks out

- [ ] Confirm cohort size for the next quarter (target: 3-4 students)
- [ ] Open the Phase 0 application form for new applicants
- [ ] Block Grey-review time for incoming applications
- [ ] Confirm Nanopore flow cell availability for the harvest week (week 3 of the program)
- [ ] Confirm PacBio HiFi capacity at UCD Genome Center if running the bonus pass

## ~6 weeks out

- [ ] Grey reviews applications, accepts cohort
- [ ] Grey assigns one mutant line per accepted student
- [ ] For each line: confirm we have viable seed, confirm genotype-validated parent stock, confirm SALK line ID + AGI locus
- [ ] Send each student their line assignment + Phase 0 checklist
- [ ] Order T-DNA genotyping primers for every assigned line (1-2 wk lead time on IDT)
- [ ] Stage frozen backup tissue from each line (genotype-confirmed, ~500 mg, labeled, in -80 with location logged in [[lab-map]])

## ~4 weeks out

- [ ] Confirm primers arrived, aliquoted, labeled per line
- [ ] Confirm backup tissue staged — log box + position in vault
- [ ] Greenhouse space allocated for the cohort (rack space, table space)
- [ ] Reagents stocked for Week 3 critical path: HMW extraction kit, Qubit reagents, gel reagents, library prep kit, flow cells
- [ ] Pipette tips, tubes, gloves stocked for cohort size
- [ ] Confirm autoclave + clean bench access scheduling won't bottleneck

## ~2 weeks out

- [ ] Each student should have completed online safety training by now — confirm
- [ ] Each student schedules in-person safety with Kehan
- [ ] Each student schedules greenhouse training (if not already)
- [ ] Lab notebook templates ready in the handbook (notebooks app)
- [ ] Bulk frozen practice tissue staged for Week 1-2 extraction drills

## ~1 week out

- [ ] Pipette school stations set up
- [ ] Agar reagents pre-weighed for Day 1 (or solution pre-prepped for student to dilute)
- [ ] Stock seed packets labeled and ready for Day 1 planting
- [ ] Germination assay plates prepped (one set per student, includes their line + comparison stocks)
- [ ] Day 1 schedule sent to each student

## Day 1

- [ ] Kehan walks each student through pipette school
- [ ] Each student plants their line in the greenhouse
- [ ] Each student plates germination assay
- [ ] Lab notebook started (one entry per student)

## After Week 3 (sequencing submitted)

- [ ] Confirm sequencing run started
- [ ] Track expected data return date per student
- [ ] Block lab Mac for assembly runs in Weeks 4-7

## After Week 10 (cohort wraps)

- [ ] Each student gives lab-meeting presentation
- [ ] Grey + Kehan grade against [[capstone-rubric]]
- [ ] Decide tech / project-path offers for students who hit all criteria
- [ ] Retro: what to fix for next cohort
- [ ] Update this checklist with anything that bit us

## Standing inventory targets (for a 3-4 student cohort)

| Item | Target stock |
|---|---|
| Nanopore flow cells | 4-6 |
| HMW extraction kits | 8 reactions per student × cohort size, plus 25% buffer |
| Library prep kits | 2 reactions per student × cohort size |
| Qubit dsDNA HS reagent | enough for ~30 measurements per student |
| Stock T-DNA primers (per assigned line) | 100 µM working stock, 4 aliquots per student |
| Frozen backup tissue (per line) | ~500 mg, vacuum-sealed, -80°C |
| Gel comb sets, agarose | enough for 2-3 PCR runs per student |

## Failure modes we plan for

- **Student's HMW extraction fails twice** → use backup tissue, document the pivot
- **T-DNA PCR shows wild-type contamination** → see [[contamination-escape]]
- **Nanopore run yields low data** → re-load the flow cell or re-prep library; if that fails, escalate to Grey for either a re-extraction or pivoting to PacBio HiFi
- **Student misses training deadlines in Phase 0** → Day 1 slips; do not start them on a partial Phase 0
- **Greenhouse space tight** → growth chamber backup (Robbins 160 or Amundson 11), documented in the [[lab-map]]
