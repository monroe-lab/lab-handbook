# Sierra Gold Mother Tree Leaf Tubes (2026-05-01)

Bag of 3 × 50 mL falcon tubes of leaf tissue collected from the three Sierra Gold Nurseries mother trees that the TC lines descend from.

- **Tubes (each labeled and dated 05/1/2026):**
  - `FRP-Clonal UCB1` — mother tree planted 2019
  - `Platinum` — mother tree planted 2020
  - `Yankee` — mother tree planted 2021
- **Source:** Sierra Gold Nurseries (Micah Stevens, Reid Robinson). Delivered with the first TC sample batch by Steven Lee / Micah, week of 2026-05-04.
- **Storage:** [[locations/shelf-minus80-a-1|Shelf 1]] of [[locations/freezer-minus80-a]] (top shelf), in a bag.
- **Provenance note:** Any TC line predating its mother-tree planting date came from a prior mother tree Micah no longer has access to. So these three leaf samples ground-truth the genetic source for: Yankee-2021 onward, Platinum-2020 onward, FRP Clonal UCB-1-2019 onward.
